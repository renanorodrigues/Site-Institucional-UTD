Iniciação a Programação

O que é

Iniciação a programação é o modo como se escreve um programa de computador, um algoritmo. Um algoritmo é uma sequência de passos para se executar uma função. Um exemplo de algoritmo, fora da computação, é uma receita de bolo. E assim como uma receita, tudo precisa ser seguido para o código funcionar.

Com esse curso você vai

  • Dar os seus primeiros passos de maneira prática!
  • Iniciar na programação JavaScript!
  • Entender as estruturas fundamentais de programação

Pré-requisitos de seleção

  • Ser alfabetizado;
  • No mínimo ter 16 anos;
  • A quem tem conhecimento prévio do assunto
Horários

Manhã: 08h às 10h.
Tarde: 15h às 17h.

Conteúdo programático do curso

Carga Horária: 60 Horas/Aula

Módulo I

Conceitos Gerais de Lógica
  • Fundamentos de lógica aplicados a vida real
  • Conceituação de lógica na programação de computadores

Módulo II

Introdução a Lógica com Ferramenta Gráfica
  • Apresentação da Ferramenta
  • Tipos de Dados com Ferramenta Gráfica
  • Uso de Operadores Lógicos e Matemáticos
  • Estruturas de Controle com Ferramenta Gráfica
  • Criação de Procedimentos

Módulo III

Introdução a Lógica em Linguagem de Computadores
  • Abordagem sobre linguagem para programação de computadores
  • Caracterização da Linguagem de Programação
  • Tipos de Dados em Linguagem de Programação

Módulo IV

Introdução a Banco de Dados
  • Conceitos Gerais de SGBD
  • Criação de Tabelas e Manipulação de Registros
  • Chaves Primárias, Secundárias e Campos Únicos
  • Comandos: Inserção, Atualização e Remoção de Registros

A UTD

A Universidade do Trabalho Digital - UTD é uma iniciativa do Governo do Estado, através da Secretaria da Ciência, Tecnologia e Educação Superior (SECITECE) que desenvolve as suas atividades em parceria com Instituto Centro de Ensino Tecnológico (CENTEC). Com cursos que vão desde uma formação básica até níveis mais avançados, a UTD proporciona formação gratuita na área de Tecnologia da Informação e Comunicação (TIC), como forma de melhorar a qualificação profissional em áreas estratégicas, promovendo o desenvolvimento sustentável do Estado e facilitando a inserção no mercado de trabalho.
